`Tomato` 是一个番茄工作法的小工具。

## 开始

### 安装

这个项目使用 [go](https://golang.org)开发，如果你本地已经安装了，可以使用以下命令安装。

```sh
$ go get github.com/go-gosh/tomato/cmd/gomato
```

你也可以在 [Release](https://github.com/go-gosh/tomato/releases) 找到编译好的二进制文件。

### 使用说明

```sh
$ $GO_ROOT/bin/gomato
```
